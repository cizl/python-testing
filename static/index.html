REST Endpoints

POST '/api/store/string:name - create store with specified name

GET '/api/store/string:name' - retrieve store with specified name

DELETE '/api/store/string:name' - delete store with specified name

POST '/api/item/string:name' - create item with specified name

GET '/api/item/string:name' - retrieve item with specified name

DELETE '/api/item/string:name' - delete item with specified name

GET '/api/items' - get the list of items

GET '/api/stores' - get the list of stores

POST '/api/register' - register user
